# expenses-crawler
## 배경
공무원 업무 추진비 내역은 부처마다 다른 사이트와 다른 경로에 업로드 되고 있음

고로, 공무원 맛집 지도의 데이터 수집을 하려면 수동으로 해야하는 부분이 있음

이러한 문제를 해결하기 위해서 제작
## 목적
각 부처별 업무 추진비 내역을 크롤링으로 다운로드 받을 수 있는 크롤러 개발
